## RobotDreams Node.js course

Task app - project

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Build and run the DB

```bash
$ docker compose up --build
```

## Run tests

```bash
# unit tests
$ npm run test
```
